    Bonfire.Translation.Behaviour behaviour (Bonfire v1.0.2-social-alpha.37)

    View Source

    Behaviour for translation adapters.

    Adapters must implement these callbacks to provide translation services. All language codes should be normalized to lowercase ISO 639-1 format (e.g., "en", "fr", "de").

    Usage

    To create a translation adapter, implement this behaviour:

    defmodule MyApp.Translation.MyAdapter do
  @behaviour Bonfire.Translation.Behaviour

  @impl true
  def translate(text, source_lang, target_lang, opts) do
    # ...
  end

  # ... other callbacks
end

    Summary

    Callbacks

    available?()

    Checks if the translation service is available and configured.

    detect_language(text)

    Detects the language of the given text. Returns the detected language code and confidence score (0.0 to 1.0).

    supported_languages()

    Returns list of supported languages with their available target languages.

    supports_pair?(source_lang, target_lang)

    Checks if translation between source and target language is supported.

    translate(text, target_lang, opts)

    Translates text to the target language, auto-detecting the source language.

    translate(text, source_lang, target_lang, opts)

    Translates text from source language to target language. Pass nil as source_lang to auto-detect.

    translate_batch(texts, source_lang, target_lang, opts)

    Translates multiple texts at once. More efficient for batch operations.

    translation_adapter()

    Declares a translation adapter module
