    Bonfire.Notify

    Copy Markdown View Source

    An extension for Bonfire that handles:

    • Records user notifications
    • Sends web push notifications

    Configuration

    For Web Push, you need to generate keys: Bonfire.Notify.WebPush.generate_keys_env()

    Then set these env variables: WEB_PUSH_PUBLIC_KEY VAPID public key. WEB_PUSH_PRIVATE_KEY VAPID private key.
