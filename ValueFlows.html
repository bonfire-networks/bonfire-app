        View Source ValueFlows (Bonfire v0.9.10-classic-beta.127)

        ValueFlows is a Bonfire extension that implements ValueFlows for federated networks to conduct economic activities.

        Implemented so far:

        • Intent
        • Proposal
        • Economic Event
        • Types of Action
        • Resource Specification
        • Economic Resource
        • Process Specification
        • Process
        • Claim (WIP)
        • Value Calculations (WIP)

        As well as an optional GraphQL API for all of the above...
