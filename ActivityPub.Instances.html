    get_or_discover_signature_format(uri)

    Returns the signature format for a host, running discovery first if needed.

    Checks cache first, then runs discovery (WebFinger, nodeinfo, FEP-844e) if the format is unknown and discovery hasn't been attempted recently. Always returns :rfc9421 or :cavage.
