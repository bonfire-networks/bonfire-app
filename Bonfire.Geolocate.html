    Bonfire.Geolocate (Bonfire v1.0.0-social-rc.2)

    View Source

    An extension for Bonfire that handles:

    • Geolocation schema
    • Optional GraphQL API for geolocations
    • Geocoding of addresses
    • Reverse geocoding
