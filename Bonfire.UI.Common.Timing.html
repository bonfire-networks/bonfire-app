    Bonfire.UI.Common.Timing (Bonfire v1.0.2-social-alpha.29)

    View Source

    Macros for instrumenting code blocks with Server-Timing metrics.

    When PAGE_PROFILER_ENABLED is not set at compile time, the macros compile to just the block — zero overhead in production.

    When enabled, records timing to the process dict via ServerTimingPlug.

    Usage

    import Bonfire.UI.Common.Timing

# Single measurement (overwrites previous value for the same key):
time_section :my_operation do
  expensive_work()
end

# Accumulating measurement (adds to previous value for the same key):
time_section_accumulate :maybe_component do
  work_called_many_times()
end

    Summary

    Functions

    time_section(key, list)

    Instruments a block, recording wall-clock time under key. Overwrites any previous value for the same key.

    time_section_accumulate(key, list)

    Like time_section/2, but accumulates time across multiple calls for the same key.
