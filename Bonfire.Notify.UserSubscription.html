    parse_subscription_data(params)

    Parses browser subscription data into the format expected by our schema. Returns an error tuple if the data structure is invalid.

    Supports both standard browser format and Mastodon API format:

    • Browser: %{"endpoint" => "...", "keys" => %{"p256dh" => "...", "auth" => "..."}}
    • Mastodon: %{"subscription" => %{"endpoint" => "...", "keys" => %{...}}, "data" => %{"alerts" => %{...}, "policy" => "..."}}
