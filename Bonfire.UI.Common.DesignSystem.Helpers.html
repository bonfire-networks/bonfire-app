    Shared helper functions for design system components. These handle common patterns like Phoenix LiveView attribute conversion.

    Summary

    Functions

    phx_value_attrs(values)

    Convert a map of phx_values to phx-value-* attributes. Uses string keys to avoid atom table exhaustion from dynamic input.

    rate_limit_attrs(throttle, debounce)

    Build rate limiting attributes (throttle or debounce) as a list. Returns an empty list if neither is set.
