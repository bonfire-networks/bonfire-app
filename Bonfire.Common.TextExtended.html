        blank?(xA)

        View Source

        Returns the length of the input based on its type.

        Examples 

        iex> strlen("hello")
5

iex> strlen([1, 2, 3])
3

iex> strlen(%{})
0

iex> strlen(nil)
0

iex> strlen(0)
0

iex> strlen(123)
1
